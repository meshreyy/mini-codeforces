# Merge Sort

## Problem Statement
Given an array of integers, sort it in ascending order using merge sort.

## Input Format
- First line: integer `n` (size of array)
- Second line: `n` space-separated integers

## Output Format
Print the sorted array space-separated.

## Sample Input
```
5
4 2 7 1 9
```

## Sample Output
```
1 2 4 7 9
```
