# Binary Search

## Problem Statement
Given a sorted array and a target value, find the index of the target using binary search.

## Input Format
- First line: integer `n` (size of array)
- Second line: `n` space-separated sorted integers
- Third line: integer `target`

## Output Format
Print the index of the target, or -1 if not found.

## Sample Input
```
5
1 3 5 7 9
7
```

## Sample Output
```
3
```
